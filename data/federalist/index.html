Each of the 85 Federalist Papers are stored in a separate text file under this directory:

  • 1.txt
  • 2.txt
  • 3.txt
  • ...
  • 85.txt

The authors of each document are stored in the following CSV file, also under this directory:

  • authorship.csv

Note that 15 of these papers do not have an author listed. The authorship of these papers is disputed or unknown. The rest are attributed to either Alexander Hamilton, John Jay, or James Madison.
