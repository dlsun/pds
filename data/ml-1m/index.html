MovieLens Data Set

The following data comes from the MovieLens 1M data set collected in 2003 by the GroupLens research group at the University of Minnesota.

It consists of three files:

  • Movies: https://dlsun.github.io/pods/data/ml-1m/movies.dat
  • Ratings: https://dlsun.github.io/pods/data/ml-1m/ratings.dat
  • Users: https://dlsun.github.io/pods/data/ml-1m/users.dat

Note that the values are delimited by ::. The column names are also missing from the files. For more information about the data set, in particular the column names, refer to the README file.
